# Tópicos abordados no curso

# Usando AAA
	 - Arange - Act -  Acert  

	 Arange: Inicializa as variáveis.
	 Act: Invoca Metodos para testar.
	 Acert: verifica ação.

# Boas Praticas

	 Evitar códigos grande e complicados.
	 Utilize constantes quando possivel.
	 Armazene informações no arquivo .config.
	 faça teste unitários bem perto do código criado.
